
## Добавление адреса криптокошелька на страницу депозита

На странице депозита перед формой будет добавлен блок с адресом кошелька для перевода средств, чтобы пользователи видели куда отправлять USDT.

### Что будет сделано

1. **Блок с адресом кошелька** -- над формой появится карточка с адресом `0x66E8b231d0B935cEc2327Ce5bedBA5a00E230aF3`, кнопкой копирования и иконкой.

2. **Переводы** -- добавлю ключи локализации для заголовка блока и уведомления о копировании в `LanguageContext.tsx`.

### Технические детали

- В `src/pages/Deposit.tsx` добавить блок перед `motion.form` с адресом кошелька и кнопкой "Скопировать" (используя `navigator.clipboard.writeText`).
- В `src/contexts/LanguageContext.tsx` добавить ключи:
  - `deposit.wallet_address` (ru: "Адрес для перевода", en: "Wallet Address for Transfer")
